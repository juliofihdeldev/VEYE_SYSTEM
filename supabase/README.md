# Supabase (Edge Functions)

This folder is managed by the [Supabase CLI](https://supabase.com/docs/guides/cli).

## Useful commands (from repo root)

| Command | Purpose |
|---------|---------|
| `supabase start` | Local Postgres, Auth, Studio, Edge runtime (Docker) |
| `supabase functions serve` | Serve Edge Functions locally |
| `supabase functions deploy telegram-monitor` | Deploy one function to the linked project |
| `supabase link` | Link CLI to your hosted Supabase project |

## Functions

- `telegram-monitor` — HTTP placeholder for the Telegram monitor migration. See `docs/SUPABASE_EDGE.md`.
